# Karmath
## Dependencies
Dependencies installation script ( Unix )
```sh
sudo apt install python3 pip3 python3-venv
git clone --recursive git@github.com:ShisuiMadara/HackIISC.git
cd HackIISC
python3 -m venv venv
source venv/bin/activate
pip install -r ./requirements.txt
```
